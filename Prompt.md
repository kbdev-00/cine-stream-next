# 🤖 AI Prompts Used – Cine-Stream Next.js Movie Application

This document lists the AI prompts used during the development of the **Cine-Stream Movie Application built with Next.js 15**.
AI assistance was used for learning concepts, architectural guidance, and debugging while implementing **Server-Side Rendering, dynamic routing, and SEO optimization**.

---

# 📌 Prompt 1 – Understanding Server Components

"Explain the difference between Client-Side Rendering (CSR) and Server-Side Rendering (SSR) in Next.js and when to use Server Components."

Purpose:
To understand how Next.js renders pages on the server to improve performance and SEO compared to traditional React applications.

---

# 📌 Prompt 2 – Fetching API Data in Next.js

"How can I fetch movie data from an external API inside a Next.js Server Component without using useEffect?"

Purpose:
To implement server-side fetching of popular movies from the **TMDB API** on the home page.

---

# 📌 Prompt 3 – Building Reusable Components

"How to create a reusable MovieCard component in Next.js that receives movie data via props?"

Purpose:
To display movie posters, titles, and links using reusable components in a responsive grid layout.

---

# 📌 Prompt 4 – Next.js File-Based Routing

"How does file-based routing work in the Next.js App Router and how can I create routes for different pages?"

Purpose:
To understand how the **app folder structure automatically creates routes** without using React Router.

---

# 📌 Prompt 5 – Dynamic Routes

"How can I create a dynamic route in Next.js like /movie/[id] to display details of a selected movie?"

Purpose:
To build a **movie details page** that dynamically loads data for each movie using its unique ID.

---

# 📌 Prompt 6 – SEO Metadata Optimization

"How to dynamically generate page metadata in Next.js using generateMetadata for better SEO?"

Purpose:
To dynamically update the **title and meta description** of each movie page based on its data.

---

# 📌 Prompt 7 – Image Rendering & Optimization

"What is the difference between using the Next.js Image component and the standard img tag?"

Purpose:
To understand optimized image loading and improve performance for movie posters.

---

# 📌 Prompt 8 – Next.js Project Structure

"What is the recommended folder structure for a scalable Next.js application using the App Router?"

Purpose:
To organize the project with separate folders for **components, pages, and API integrations**.

---

# ⚠ Disclaimer

AI tools were used for **learning guidance, debugging, and concept clarification**.
All implementation decisions, UI development, and integration logic were manually reviewed and implemented during the development of this project.
